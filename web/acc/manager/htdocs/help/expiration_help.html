DATE d'EXPIRATION - EXPIRATION DATE
Cet attribut définit la date d'expiration du compte. Le format est "jour mois année" (ex: 20 april 2002).
Les mois en anglais sont : january, february, march, april, may, june, july, august, september, october, november, december
This attribute defines the user expiration date. It should be in the format "day month year" like: "20 april 2002"

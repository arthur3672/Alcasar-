VOLULE MAX d'OCTETS TRANFERE - MAX TOTAL OCTETS
Cet attribut définit le nombre d'octets maximum que l'usager est autorisé à transférer (somme des octets émis et reçus). Après cette limite l'usager est déconnecté.
This attribute defines the maximum number of octets the user is allowed to transfer (sum of octets transmitted and received). After this limit has been reached the user will be disconnected.
